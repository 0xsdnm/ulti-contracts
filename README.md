## Usage

Format files:
`yarn format`

Lint files:
`yarn lint`

Compile files:
`yarn compile`

Run tests:
`yarn test`

Deploy contract to bsc testnet:
`yarn deploy-testnet`  - put your mnemonic to `.secrets.json`

Verify contract on bscscan:
`yarn verify-testnet CONTRACT_ADDRESS` - put your bscscan api key to `.secrets.json`
